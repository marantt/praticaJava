# PraticaJava

## Descrição

Resolução de exercícios e práticas desenvolvidas em Java para reforçar conceitos fundamentais da linguagem e da programação orientada a objetos.

## Tecnologias

- Java

## O que foi desenvolvido até agora

- Estruturas de controle
- Métodos
- Classes e objetos
- Lógica de programação

## Objetivo do Projeto

Aprimorar o raciocínio lógico e a base em Java por meio de exercícios práticos.
